# Сайт домашних работ для класса или кружка

## Функции

- Возможность регистрироваться с почтой и паролем и иметь статус учителя или ученика
- Добавлять, редактировать, устанавливать определенных учеников для их выполнения, прикреплять файлы и удалять домашние задания за учителя
- Просматривать ближайшие домашние задания за ученика, скачивать прикрепленные материалы

## Архитектура проекта

- bd - папка, содержащая базу данных сайта
- data - папка с программами, содержащими blueprint-функции и классы для управления БД
- static - папка с файлами и css оформлением сайта
- templates - папка с html структурой сайта
- main.py - главная функция, запускающая весь сайт
- .gitignore
- README.md
- requirements.txt - файл с названиями и версиями всех нужных библиотек

## Структура БД

- users - таблица с информацией об аккаунтах, включаящая в себя id, имя, статус(Учитель или ученик), почту и пароль.
- homeworks - таблица с домашними заданиями, предмете задания, информирующая о самом задании, учителе, учениках, которые должны его сделать, дате и ссылкой на прикрепленный файл.
- subjects - таблица с id и названиями предметов, используемая таблицей homeworks

## Дизайн сайта

- Верхняя полоска - название сайта слева, справа - кнопки "Новое ДЗ"(У учителя) или "Регистрация" и "Вход", если не зашли. Если зашли, то отображается имя зарегестрированного, при нажатии на которого можно выйти из аккаунта
- Остальная часть сайта - Здесь почти все поле, преимущественно в центре, расположены карточки домашних заданий в приблизительно таком формате:
[][][]
[][][]
каждая карточка имеет сверху вниз информацию о: задании, учителе, задавшем задании, сроке сдачи, предмете и файле к заданию, если учитель прикрепит.
- *!!! Демонстрируются не все карточки ДЗ, а только те, чей дедлайн истекает через неделю(если у всех) или через 2 недели(если у определенной группы)*

## Инструкция по установке

- Установите архив репозитория и распакуйте в отдельную папку
- Откройте терминал и через cd перейдите в папку репозитория
- Создайте виртуальное окружение отедльно в эту папку следующей командой: ```python -m venv venv```
- Установите все библиотеки из requirements.txt следующей командой: ```pip install -r requirements.txt```
- Для запуска проекта используйте main.py: ```py main.py```
